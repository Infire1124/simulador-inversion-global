# Calculadora Financiera Dinámica

Esta es una herramienta web diseñada para realizar cálculos y proyecciones financieras de forma dinámica y eficiente.

## ¿Qué resuelve esta aplicación?
Esta calculadora permite a los usuarios:
* Realizar proyecciones de su estado patrimonial.
* Obtener resultados financieros rápidos.
* Visualizar el impacto de diferentes variables en sus finanzas personales.

## Tecnologías utilizadas
* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Chart.js
